# Практическое задание №2

## Цель задания
Разработать приложение на языке программирования Python для выявления сетевых аномалий.

## Требования к приложению
1. **Функция захвата данных**: Приложение должно иметь возможность захватывать данные в реальном времени. Обработка данных должна быть потоковой, то есть мы не используем заранее захваченные данные.
2. **Функции анализа и выявления сетевой аномалии**: Приложение должно анализировать захваченные данные и выявлять сетевые аномалии по установленным правилам. Эти правила должны быть обоснованы.
3. **Функция эмуляции сетевой аномалии**: Приложение должно иметь возможность эмулировать сетевую аномалию для тестирования функций обнаружения сетевых аномалий*.

## Требования к коду
- Код должен быть написан на языке программирования Python с применением сетевых библиотек.
- Код должен быть выложен на GitHub. Если код будет выложен одним коммитом, будут дополнительные вопросы.

## Защита проекта
- Необходимо защитить свое приложение и ответить на поставленные вопросы. Все приложения будут проверены на работоспособность.

## Требования к проекту
- Проект должен включать в себя файл с постановкой задач для разработки приложения.
- Проект должен включать в себя файл с аннотацией работы приложения (с использованием описания алгоритмов в PlantUML).

## Дедлайн защиты
31 мая или 5 июня.
